Which Browser?

The preferred browser for viewing these pages is Netscape Navigator, version 3.0 or above.

The original version of these pages made heavy use of JavaScript. Unfortunately, MicrosoftTM®© Internet ExplorerTM®©'s version of JavaScript could not handle important constructs such as include files.

I spent a very unpleasant forty-five minutes modifying all my files to avoid using this feature, but the pages now work with Internet ExplorerTM®© version 3, which many people still have.

What this means to you is that one or two of the pages will take a bit longer to load than in the original version.

What it means to me, as the author, is that any change I would have been able to make in one file now must be made in several files, with the added possibility of errors. It also means that my structure for navigating between pages is no longer as general and easy to modify.

So congratulations to Netscape for providing a powerful tool like Javascript. Shame on MicrosoftTM®© for not supporting Javascript's best features in their browser.
